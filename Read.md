# Login WebAPI
## Simple App with .NET Core 3.1 to shows how the auth with JWT works

### Folders Structure
- Domain
    * Commands
    * Entities
    * Enuns
    * Handlers
    * Queries
    * Respositories

- API
   * Controllers

- Infra
   * Respository
   * Contexts
   * Settings

- Tests
   * Mocks
   * Fakes

The project its a basic sample of auth with JWT, using .NET Core and SQL Server, using the basics principles of DDD, Clean Code and SOLID.
